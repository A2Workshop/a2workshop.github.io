Guía de instalación

INTRODUCCIÓN

Como muchos saben, el doblaje al español latino nunca existió para este juego, este mod implementa el doblaje realizado en la película lanzada el mismo año creada a partir de las cinemáticas del juego.

En esta guía se explicará el proceso de instalación del doblaje y traducción. Esta guía NO es para obtener material protegido por derechos de autor de forma ilegítima.

TRÁILER

DESCARGAS

Requerimientos:

  • Algún programa para descomprimir archivos 7z (7-Zip, NanaZip, etc)
  • QuickBMS: https://aluigi.altervista.org/papers/quickbms.zip
  • ttgames.bms: https://aluigi.altervista.org/bms/ttgames.bms

Enlace del mod:

INSTALACIÓN

  • Localiza la carpeta raíz del juego. (Ej. C:\Program Files (x86)\Steam\steamapps\common\LEGO Batman 2)
  • Descomprime el contenido de quickbms.zip, el archivo Extract_LB2_Files.bat de LB2EspLatinoModx.x.7z en la carpeta del juego y mueve el archivo ttgames.bms.
  • El juego debe estar configurado en el idioma Español (México) para funcionar correctamente, para esto, hay 3 opciones:
      • -> (Windows 10/Windows 11) Cambiar el formato regional a Español (México) en la configuración de Windows.

      -> (Windows) Ejecuta el archivo ForceLB2SpanishMex.reg que viene incluido con el mod. (Requiere permisos de administrador)


      -> (Steam Deck/Linux) Agregar la siguiente línea a las configuraciones de lanzamiento del juego: HOST_LC_ALL=es_MX.UTF-8 %command%
  • Ejecuta Extract_LB2_Files.bat para extraer los archivos .DAT del juego.
  • Extrae el contenido de LB2EspLatinoModx.x.7z en la carpeta del juego y reemplaza los archivos.

    • LIMITACIONES

    Este mod fue desarrollado a partir de las voces de la película "Batman LEGO: La unión de los superhéroes", y por lo tanto, tiene algunas limitaciones:

    • Algunas líneas pueden no estar dobladas debido a limitaciones del material original y se silenciarán, reemplazarán por otras similares o quedarán en castellano dependiendo del caso.
    • La calidad técnica de las voces puede variar, ya que no se cuenta con los archivos de audio originales de voz.

    CRÉDITOS

    • Doblaje original de la película.
